# Password Generator (CLI)

## Overview
This project is a command-line password generator developed in Python. It creates strong and random passwords using a combination of uppercase letters, lowercase letters, numbers, and special characters.

## Features
- Generates secure random passwords
- Uses Python's random module
- Includes uppercase letters
- Includes lowercase letters
- Includes numbers
- Includes special characters
- User-defined password length
- Input validation

## Technologies Used
- Python 3

## Project Structure
- task4.py

## How to Run
1. Download or clone the repository.
2. Open the project in VS Code.
3. Run the following command:

```bash
python task4.py
```
